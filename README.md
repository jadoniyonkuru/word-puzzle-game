# Word Puzzle Game

A fun and interactive word search puzzle game built with HTML, CSS, and JavaScript.

## Features

- **Multiple Levels**: 4 progressive levels with increasing difficulty
- **Word Search Mechanics**: Find words horizontally, vertically, and diagonally
- **Interactive Gameplay**: Click and drag to select words
- **Scoring System**: Points based on word length and completion time
- **Timer**: 60-second countdown with bonus time for completing levels
- **Hints**: Get hints when stuck (costs points)
- **Pause/Resume**: Pause the game at any time
- **Responsive Design**: Works on desktop and mobile devices
- **Visual Feedback**: Animations and color changes for found words

## How to Play

1. **Find Words**: Look for the hidden words listed on the right side of the screen
2. **Select Words**: Click and drag across letters to select a word
3. **Word Directions**: Words can be found in 8 directions:
   - Horizontal (left to right, right to left)
   - Vertical (top to bottom, bottom to top)
   - Diagonal (all 4 diagonal directions)
4. **Complete Levels**: Find all words to advance to the next level
5. **Beat the Clock**: Complete all words before time runs out!

## Game Levels

- **Level 1**: Simple 3-letter animal words
- **Level 2**: 5-6 letter fruit words
- **Level 3**: Computer-related terms
- **Level 4**: Programming and development terms

## Scoring

- **Word Points**: 10 points per letter in found words
- **Level Completion**: 100 bonus points
- **Game Completion**: 500 bonus points
- **Hint Penalty**: -5 points for using hints

## Controls

- **New Game**: Start a fresh game
- **Hint**: Highlights an unfound word (costs 5 points)
- **Pause/Resume**: Pause the timer and gameplay

## Technical Details

- **Grid Size**: 10x10 letter grid
- **Word Placement**: Automatic random placement in valid directions
- **Mobile Support**: Touch events for mobile devices
- **Browser Compatibility**: Works in all modern browsers

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser
3. Start playing immediately - no installation required!
